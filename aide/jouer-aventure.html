La pge d'aventure

Lorsque vous cliquez sur le lien vers une aventure, vous arrivez sur une page speciale pour cette aventure. Vous remarquerez que, sous le titre du site, est apparu, en bleu, le titre ce cette aventure. Le menu a egellement change. Vous y trouverez les liens pour:

  • Retourner sur le site, a l'accueil
  • Aller sur la page "introduction et options", celle ou vous ariverez en cliquant sur le lien de l'aventure( en savoir plus: lien "introduction" du menu "aventures")
  • Les liens d'inscription/connexion, ou de deconnexion
  • Et si vous etes l'auteur de cette aventure, un lien pour la modifier
Pour continuer dans l'aide, accedez a la page "introduction".

Un poiint important est le stockage de votre partie en cours.
Si vous etes inscrit et connecte: Votre partie est stockee sur votre compte, vous pouvez y acceder depuis n'import ou, vous ne risquez pas de la perdre.
Si vous n'estes pas connecte: Votre partie est stockee sur votre navigateur, via des petits fichiers nomes cookies. Si vous les supprimer, ou videz les donnees du navigateur, vous perdez toutes vos parties.

Si vous vous inscrivez, vous pouvez recuperer les parties comencees avant pour les tranferer sur votre compte.
